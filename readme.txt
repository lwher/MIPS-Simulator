实现一个支持所有整数操作的MIPS解释器，并模拟五级流水工作原理

具体实现要求见mips-simulator.pdf

SPIM.exe即为解释器(编译环境为windows，其他平台用户可自行编译文件夹SPIM中的源码)

使用方法：在当前目录下打开控制台，输入命令 SPIM + 空格 + xxxxx.s
(其中，xxxxx.s即为要运行的mips源码)

测试数据：
https://github.com/lwher/mips-simulator-testcases

checker.exe为测试脚本，讲其与SPIM.exe一起放在数据以及输入输出所在目录，在命令行中运行即可知道结果。
